# quantum-algorithms-condensed-matter
Quantum computing meets condensed matter physics. Seven Jupyter notebooks progressing from NISQ fundamentals (Grover, VQE, QAOA, Krylov) to a research application: finding the BCS superconducting gap and FFLO order parameter via Grover-based optimisation of the free energy density. Simulated on AaltoQ20-scale hardware using Qiskit.
